## Overview

<a href="#">Pure Paste</a> is a lightweight macOS menu bar utility developed by Sindre Sorhus that solves one of the most persistent annoyances in daily Mac use: unwanted rich text formatting carried over when copying and pasting content between applications. When you copy text from a website, email, PDF, or styled document and paste it into another application, the clipboard typically brings along fonts, colors, sizes, hyperlinks, and hidden formatting metadata that clashes with the destination's style. Pure Paste intercepts this process and strips all formatting automatically, delivering clean plain text every time you paste.

The <a href="#">automatic stripping mode</a> works transparently in the background without any keyboard shortcut changes or workflow adjustments. Copy text from anywhere using the standard Command-C shortcut, and Pure Paste silently cleans the clipboard contents. When you paste with Command-V, the text arrives already stripped of all rich formatting — no need to remember an alternative paste shortcut, no need to run a secondary cleaning step, and no accidental pastes of styled content that then require manual fixing. The entire process is invisible until you notice that formatting problems have simply stopped occurring.

<a href="#">Selective formatting control</a> allows fine-tuning of exactly which attributes get removed. Users who work with content that legitimately needs some formatting preserved can configure Pure Paste to retain specific attributes — keeping bold and italic emphasis while stripping colors and fonts, for example — so the tool adapts to actual workflow needs rather than applying a single rigid behavior across all contexts. <a href="#">Allowlist configuration</a> defines which formatting types survive the cleaning process, giving precise control over the clipboard transformation.

The <a href="#">menu bar interface</a> provides instant access to Pure Paste settings and a toggle to temporarily suspend automatic stripping when raw formatted content genuinely needs to be pasted. A keyboard shortcut allows quick enable and disable without touching the mouse. <a href="#">Per-application exclusions</a> exempt specific apps from automatic stripping, useful for applications like design tools where formatted clipboard content serves a legitimate purpose and plain text conversion would break intended workflows.

## Key Features

- <a href="#">Automatic plain text conversion</a> strips all rich formatting from clipboard content silently every time you copy without changing any shortcuts
- <a href="#">Zero workflow disruption</a> works with standard Command-C and Command-V so existing habits require no adjustment whatsoever
- <a href="#">Selective attribute filtering</a> configures which formatting types are removed keeping bold italic or other specific attributes when needed
- <a href="#">Allowlist for formatting types</a> defines precisely which clipboard attributes survive cleaning for fine-grained workflow control
- <a href="#">Temporary disable toggle</a> suspends automatic stripping instantly via menu bar or keyboard shortcut when formatted paste is needed
- <a href="#">Per-application exclusions</a> exempt chosen apps from clipboard cleaning so design and layout tools receive formatted content unmodified
- <a href="#">Hidden metadata removal</a> clears invisible tracking data and embedded source information that copies silently attach to clipboard text
- <a href="#">Hyperlink stripping</a> removes embedded URLs from copied text so link-heavy web content pastes as readable plain words
- <a href="#">Minimal resource footprint</a> runs as a menu bar agent consuming negligible CPU and memory during continuous background operation

## Additional Information

Pure Paste reflects a design philosophy common to Sindre Sorhus utilities: identify a specific friction point in daily Mac use and eliminate it completely with a tool that requires no ongoing attention after initial configuration. The application installs, gets configured once, and then operates silently for months and years without requiring any interaction. Users who spend significant time moving text between web browsers, email clients, word processors, and code editors will recognize immediately how much time and frustration automatic plain text pasting eliminates.

The application is available through the Mac App Store, ensuring straightforward installation and automatic updates. Its minimal footprint makes it compatible with any Mac workflow regardless of which other applications are in use.
